Sobre o Projeto

Este projeto consiste na geração de energia á base de hidrogênio. A ideia é mostrar a conversão do hidrogênio para eletricidade, como fonte mostrando como é possivel ser uma alternativa ao invés do combústivel fossil. Isso mostra também mostra o conceito de tecnologia verde e sendo uma solução limpa e sustentável de conversão de energia.

Objetivo

O objetivo deste projeto é desenvolver um protótipo prático de conversão de energia para o hidrogênio, promovendo o uso de energia limpa e buscando por soluções sustentáveis e inovadoras no lugar do combústivel fossil que tem malefícios para o planeta.
